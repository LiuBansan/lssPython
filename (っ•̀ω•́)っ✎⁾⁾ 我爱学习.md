# python-note


### anaconda 基本使用
- anaconda主要是一个虚拟环境管理
- 还是一个安装包管理器
- conda list: 显示anaconda安装包
- 从打恶女list：显示anaconda的虚拟环境列表
- conda create -n xxx python=2.6: 创建python版本为3.6的虚拟环境，名称为xxx
